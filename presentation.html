Introducing Atlas

O'Reilly Atlas is a tool for creating and publishing beautiful technical content.

Use your mouse or arrow keys to navigate through the rest of this presentation.

Atlas is...

Atlas is a system for writing, styling, and distributing content.  You can learn about all the features at the Atlas Documentation.

 

 

Atlas embodies 3 key ideas

  • Version control using git
  • A single source of content in a semantic format
    • HTMLBook
    • AsciiDoc
    • DocBook
    • MarkDown
  • Transformation engines based on a customizable theme
    • PDF
    • EPUB
    • Mobi
    • HTML

 

Make Books

Atlas produces professional quality PDFs that can be both printed on demand (POD) and by a traditional offset processes.

 

 

Make ebooks

Atlas produces both EPUB and MOBI, the formats used for the iBooks and Kindle.

 

 

Make sites

You can build web sites like Scott Murray's Interactive Data Visualization for the Web or the Raspberry Pi Cookbook Sampler.

 

 

Make presentations

Use Atlas themes to create a presentations you can post online.

 

 

Write in an intuitive visual editor

If you don't like plain text markup, Atlas has a intuitive visual editor.

 

 

Interactive content

Embed interactive elements like, JSBins, into the web version of your book.

D3: Pie layout

 

Open source your book on GitHub

Because it's based on git, you can easily open your content for contributions on GitHub, like we've done for Etudes for Erlang.
